# The Story Nexus 1.2.1

GitHub release title: `The Story Nexus 1.2.1`

Recommended tag: `1.2.1`

## GitHub Release Body

The Story Nexus 1.2.1 is a focused quality update for local AI workflows. It makes local model selection clearer and more reliable across prompts, agents, and writing tools.

### What's Improved

- Added a local AI default model choice in AI settings, so you can decide which local model The Story Nexus should use by default.
- Prompts can now use `Local default`, letting them follow your current local AI setup without needing to edit every prompt when you switch models.
- Prompt and agent model pickers now show `Local default` alongside your available local models, making it easier to choose between automatic and specific local model behavior.
- Local model lists refresh more reliably, even if you have not configured a cloud AI provider.
- Timeline, lorebook, style, and writing workflows now resolve the selected default model more consistently.
- The built-in prompt labels now use clearer wording for local AI choices.

### Upgrade Notes

- Existing local AI setups should continue to work.
- If you use multiple local models, open AI settings after updating and choose the model you want as your local default.
- If you leave the setting on automatic, The Story Nexus will use the first available local model.
